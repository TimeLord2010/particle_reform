## 0.0.1

* Initial release of particle_reform package
* Core ParticleReform widget for breaking down widgets into pixel particles
* Multiple particle effects:
  * Scatter: Random particle dispersion
  * ScatterDisappear: Particles scatter and fade out
  * SpinningCircle: Particles arranged in a spinning circular motion
* Customizable animation duration and easing curves
* Animation speed control per effect
* Smooth transitions between formed and scattered states
* Support for continuous looping animations
